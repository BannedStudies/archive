This is the repository for the Banned Studies Archive (BSA). The BSA is a collection of syllabi pertaining to subject areas that have been under attack by right-leaning politicians. It is an archive that seeks to preserve educational and academic freedom.

If you wish to submit to the BSA, please fill out (this form)[https://forms.gle/kA7j1dDsmYUYgGkx6].

More information about this project will be released as the project develops.
